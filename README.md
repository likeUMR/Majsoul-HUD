# AI_Mahjong

`AI_Mahjong` 是一个面向 Windows 的雀魂辅助项目，整合了流量监听、对局解析、本地 HUD 展示，以及基于麻将算法后端的推荐能力。

当前仓库主要面向源码整理与便携版发布，目标是提供一套较易运行、较易分发的本地工具链。

## 功能简介

- 基于 `mitmproxy` 抓取雀魂网页端流量
- 解析对局事件并输出基础信息
- 提供本地 HUD 叠加显示
- 接入麻将算法后端，支持推荐与期望值相关分析
- 支持 Windows 便携版打包与一键启动

## 目录结构

- `Crawler/`：监听、协议解析、HUD 与推荐逻辑
- `Algorithm/`：麻将算法后端及其运行资源
- `config/`：运行配置
- `tools/`：发布构建脚本、模板与辅助工具
- `start_all.bat`：主启动入口

## 算法来源

本项目的算法后端来源于：

- [nekobean/mahjong-cpp](https://github.com/nekobean/mahjong-cpp)

在此基础上，本仓库进行了面向本项目的集成与修改，包括但不限于：

- 适配为本地可调用的后端服务
- 增加 Windows 启动与便携版打包流程
- 与抓包、HUD、推荐链路进行整合
- 补充运行时资源和发布所需结构

原始算法实现、算法说明及相关参考资料，请优先参考上游项目。
